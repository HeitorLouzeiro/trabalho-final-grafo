## Getting Started
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/HeitorLouzeiro/trabalho-final-grafo.git
   ```
2. Access the project folder in terminal/cmd
   ```sh
   cd trabalho-final-grafo
   ```

3. Run the application.
    ```sh
     java -jar trabalho-final-grafo.jar
    ```
